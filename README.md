# kinc (Kubernetes in Containers)

A tool for creating and managing local Kubernetes clusters using Apple container 'nodes'.

## Installation

```bash
go install github.com/funkolab/kinc@latest
```

## Usage

```bash
# Create a cluster
kinc create cluster

# Get clusters
kinc get clusters

# Delete a cluster
kinc delete cluster
```
